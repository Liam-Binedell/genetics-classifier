# Environment Setup (no explicit venv required):
- Follow the instructions to install uv here, https://docs.astral.sh/uv/getting-started/installation/
- Hit a cheeky `git clone`
- `cd` into the cloned directory
- `uv sync` will generate the venv and install dependencies automatically
- You will need to tell whatever IDE you use to point at the .venv that gets created to get completions for the packages
- Happy days?
